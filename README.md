## Description 
 
  This program maintains a database of Products
 

## Installation 
 
   The Program requires Inquirer 8.2.4 and sequalize 5

## Usage 


1. Clone the repo from github (link in the contact section) 
2. Run schema.sql in psql
3. Run "node server.js" in the terminal
4. run node seeds/index.js to seed database
